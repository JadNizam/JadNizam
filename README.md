<h2 align="left">💻 Featured Projects</h1>

[⚡🚗 HazardNet](https://github.com/JadNizam/HazardNet) →
Smart real-time hazard detection system using V2V and V2X communication with reinforcement learning to help autonomous vehicles detect hazards, share alerts, and improve road safety.

[🅿️🤖 3D-RL-Autonomous-Parking](https://github.com/JadNizam/3D-RL-Autonomous-Parking) →
A 3D autonomous parking system built in Unity ML-Agents where a PPO agent learns to self-park across randomized starting positions, parking targets, and dynamic parking scenarios.

[🛰️🚙 GazeboSimulation](https://github.com/JadNizam/GazeboSimulation) →
Mars rover simulation project built with Gazebo and ROS 2, focused on rover mobility, SLAM, navigation, autonomy, and testing in realistic off-road terrain environments.

[🚦🧠 HighwayEnv-V.2](https://github.com/JadNizam/HighwayEnv-V.2) →
Custom reinforcement learning traffic environment based on HighwayEnv, extended for multi-agent driving experiments involving dense traffic, intersection decision-making, and cooperative autonomous control.
